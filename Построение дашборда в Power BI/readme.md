# Выполнено

**Построен интерактивный дашборд в Power BI**

1. Первая страница отражает общую информацию о трёх сетях магазинов. Для отображения карты и расположения магазинов требуется интернет-соединение. Размер кружочка на карте зависит от суммарной прибыли конкретного магазина. При наведении курсора на кружочек магазина, будет отражаться основная информация о нём.
2. Вторая страница отражает средний и медианный размер чека каждой сети магазинов.
3. Третья страница отражает сумму продаж каждой сети магазинов в динамике + топ 5 сверху и снизу списка по количеству позиций в чеке исходя из группы товаров.
4. Четвёртая, пятая и шестая страницы отражают динамические ABC-анализы по каждой из сетей (и по каждому магазину: присутствует множественный выбор через зажатый Ctrl) в виде простой удобной таблицы.
